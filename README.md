# Peekage

A Nuxt 4 project with:
- Vue 3 (Composition API)
- TypeScript
- TailwindCSS
- Nuxt UI
- Vite (default bundler)

## Setup

Install dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

## Build

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
